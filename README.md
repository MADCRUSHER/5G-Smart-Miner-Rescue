# 5G Smart Miner Rescue System

A crisis management system using IoT, edge intelligence,
and 5G-inspired architecture.

NodeMCU Vest Firmware



Sensors:

DHT11 — temperature \& humidity

MQ2 — gas/smoke

KY093 — heart pulse

FSR — force/pressure



Output: JSON over HTTP every 3 sec
